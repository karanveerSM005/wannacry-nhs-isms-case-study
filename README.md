# WannaCry NHS ISMS Case Study

A university cyber security case study examining the WannaCry ransomware incident and its impact on NHS healthcare services.

This project applies Information Security Management System (ISMS) principles to a healthcare environment, using the WannaCry incident as the threat scenario. It considers organisational assets, threats, vulnerabilities, risk levels and appropriate security controls.

> **Academic project:** This repository contains coursework created for educational purposes. It is not an official NHS document or a real operational ISMS.

## Overview

The case study focuses on how ransomware, outdated systems, missing patches and weak network security can affect critical healthcare services.

The project explores:

* WannaCry ransomware and the EternalBlue SMB vulnerability
* Impact on NHS healthcare operations
* Asset identification and classification
* Threat and vulnerability analysis
* Risk assessment and risk register development
* Security control recommendations
* Patch management and vulnerability management
* Backup and recovery planning
* Access control and network-security improvements
* Incident response and business continuity considerations

## Case Study Scope

The case study uses a hospital healthcare environment affected by the WannaCry ransomware incident as its scenario.

It considers important assets including:

* Electronic medical records
* Patient and staff identifiable information
* Prescription records
* Appointment scheduling systems
* Clinical decision-support systems
* Medical imaging data
* Network infrastructure
* Backup systems
* Security monitoring and intrusion-detection tools
* Medical and emergency equipment

## Security Recommendations

The report recommends controls including:

* Regular operating system and software patching
* Vulnerability scanning and remediation processes
* Secure backups with regular recovery testing
* Network segmentation
* Strong access controls and least privilege
* Multi factor authentication where appropriate
* Security monitoring and intrusion detection
* Cyber security awareness training
* Incident response planning
* Regular asset reviews and risk register updates
